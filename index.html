CSS Topic - Opacity and RGBA()

Opacity

Opacity is a CSS property used to set the transparency of an object or image. It is a range from 0 to 1. 0 is fully transparent and 1 is fully opaque.

See the Pen Opacity Pen by Sidney (@sseaton6) on CodePen.

RGBA

RGBA lets us select colors based on a combination of the primary colors, red, blue, green, and the opacity. The a value, similar to opacity, is a number that controls how transparent the color is.

Here is the context on how to insert it into HTML:   rgba(255, 255, 255, opacity)

From the above you can see that values can go up to 255, values can be as low as 0. The higher the number, the more of that color is present in the color that you mix.

See the Pen Opacity and RGBA() by shadowpeak100 (@shadowpeak100) on CodePen.

Cool Examples!

Image overlay

One cool thing that you can do with opacity is an image overlay. When you hover your mouse over it, it becomes transparent so you can see what is under it.

See the Pen Image Overlay by Sidney (@sseaton6) on CodePen.

Transparency Hover Effect

Another interesting usage of opacity is the hover effect. When you drag your mouse over an image, it will either become more transparent or more opaque depending on whether you begin with a solid image. This can be useful when writing up buttons and cards.
